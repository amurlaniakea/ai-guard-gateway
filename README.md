# 🛡️ AI Guard Gateway\n![CI](https://github.com/amurlaniakea/ai-guard-gateway/actions/workflows/sonarcloud.yml/badge.svg)\n\n[![License: AGPL-3.0](https://img.shields.io/badge/License-AGPL--3.0-blue.svg)](https://www.gnu.org/software/free-software-foundation/agpl-3.0/)\n[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)\n[![FastAPI](https://img.shields.io/badge/FastAPI-0.115+-green.svg)](https://fastapi.tiangolo.com/)\n\n**AI Guard Gateway** es un proxy inverso de seguridad diseñado específicamente para proteger endpoints de Inteligencia Artificial expuestos. Actúa como un escudo inteligente entre el cliente y el modelo de IA, mitigando riesgos críticos de seguridad y asegurando la privacidad de los datos.\n\n## 🎯 Misión\nProteger la infraestructura de IA contra el abuso y la filtración de datos, implementando capas de defensa proactiva que eviten la inyección de prompts y la exposición de información sensible (PII), manteniendo la baja latencia necesaria para aplicaciones en tiempo real.\n\nPara profundizar en la base científica y la justificación técnica de este proyecto, consulte el archivo [RESEARCH.md](./RESEARCH.md).\n\n## ✨ Características Principales\n\n### 🔒 Seguridad Activa\n- **Detección de Inyección de Prompt**: Bloqueo nativo de patrones de ataque comunes (Jailbreaks, System Overrides).\n- **Redacción de PII (Privacidad)**: Enmascaramiento automático de emails, tarjetas de crédito e IPs en las respuestas del LLM.\n- **Soporte OPA (Open Policy Agent)**: Integración con el estándar de la industria para políticas de seguridad externas y dinámicas.\n\n### 🚦 Control y Gobernanza\n- **Rate Limiting Avanzado**: Implementación de Ventana Deslizante (Sliding Window) para prevenir DoS y abuso de cuotas.\n- **Autenticación Híbrida**: Soporte para API Keys y JSON Web Tokens (JWT) con gestión de roles.\n- **Aislamiento de Identidad**: Trazabilidad completa de quién accede a qué modelo y con qué permisos.\n\n### 📈 Observabilidad Profesional\n- **Métricas de Prometheus**: Exposición de contadores de peticiones, bloqueos y latencia en `/metrics`.\n- **Logging Estructurado**: Salida de logs en formato JSON compatible con Loki/ELK.\n- **Health Checks**: Endpoints de estado para orquestadores como Docker containers.\n\n## 🛠️ Stack Tecnológico\n- **Backend**: Python 3.12 + FastAPI\n- **Gestión de Dependencias**: Poetry\n- **Seguridad**: PyJWT, OPA (Rego)\n- **Monitoreo**: Prometheus Client\n- **Servidor ASGI**: Uvicorn\n\n## 🚀 Inicio Rápido\n\n### Instalación\n```bash\n# Clonar el repositorio\ngit clone https://github.com/amurlaniakea/ai-guard-gateway.git\ncd ai-guard-gateway\n\n# Instalar dependencias con Poetry\npoetry install\n```\n\n### Ejecución\n```bash\npoetry run python main.py\n```\nEl gateway estará disponible en `http://localhost:8080`.\n\n### Ejemplo de Petición Autorizada\n```bash\ncurl -X POST http://localhost:8080/v1/chat/completions \\n     -H "X-API-Key: sk-premium-67890" \\n     -H "Content-Type: application/json" \\n     -d '{"model": "gpt-4", "messages": [{"role": "user", "content": "Hola"}]}'\n```\n\n## ⚖️ Licencia\nEste proyecto está bajo la
